Para ejecutar la Api Rest se necesita lo siguiente.
1.- tener instalado el framework nodejs https://nodejs.org/es/
2.- ingresar por terminal o cmd a la carpeta contenedora del codigo fuente.
3.-ejecutar el servidor con el comando npm run dev
4.- para tener una mayor interaccion del sistema instale postman.
5.- Usted podra visitar las siguinetes urls:

post: http://localhost:3000/api/Transferir
get:  http://localhost:3000/api/Transferir
get:  http://localhost:3000/api/EstadoCuenta/all
get:  http://localhost:3000/api/EstadoCuenta/enviadas/12345
get:  http://localhost:3000/api/EstadoCuenta/recibidas/98654
get:  http://localhost:3000/api/EstadoCuenta/balance/12345
